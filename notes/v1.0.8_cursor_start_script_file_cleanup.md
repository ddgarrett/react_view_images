**User**

when are the "dev" and "dev:electron" scripts run?

---

**Cursor**


They’re **not** run by the `start` script. You run them yourself when you’re developing.

- **`dev`** – Starts the Vite dev server (hot reload for the renderer). You’d run:  
  `npm run dev`
- **`dev:electron`** – Starts Electron with the current built app. You’d run:  
  `npm run dev:electron`

So in a typical dev workflow you might:

1. Run `npm run dev` in one terminal (Vite).
2. Run `npm run dev:electron` in another (Electron).

`start` is for a full build + run + cleanup: it runs `build`, then `electron .`, then `clean:js`. It never runs `dev` or `dev:electron`.
